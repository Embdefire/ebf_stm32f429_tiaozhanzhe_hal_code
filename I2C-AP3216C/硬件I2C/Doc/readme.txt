【*】程序简介

-工程名称：AP3216C 测试例程
-实验平台: 野火 STM32 F429 开发板 
-MDK版本：5.16
-ST固件库版本：1.5.1

【 ！】功能简介：
读板载的三合一光照传感器 AP3216C 的值。

【*】注意事项：
无

【 ！】实验操作：
电脑端使用串口调试助手，选择电脑与STM32相连的COM口，设置为115200-N-8-1，
复位开发板，即可接收STM32串口发送给电脑的调试信息。
按调试信息操作即可。


【*】 引脚分配

AP3216C：
AP3216C芯片的I2C接口与STM32的I2C1相连,且已接上拉电阻。
		SCL	<--->PB6
		SDA	<--->PB7
        INT <--->PG12
												
/*****************************************************************************************************/


【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 180MHz
-APB2 = 90MHz 
-APB1 = 45MHz

C.浮点运算单元：
  不使用
